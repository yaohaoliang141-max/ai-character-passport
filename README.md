# 🎞️ AI Video Director: 你的 AI 影视分镜架构师

这是一个为硬核 AI 视频创作者量身定制的生产力工具，采用 **Apple Aesthetic** 极简设计风格。

### ✨ 核心功能
* **剧本智能拆解**：输入长剧本，自动拆解为 10-15s 分镜。
* **即梦 (Jimeng) 深度优化**：生成的提示词完美适配中文语义与电影级运镜。
* **资产版本管理**：提示词与生成的视频片段在本地 IndexedDB 一一对应，永不丢失。
* **隐私优先**：用户自备 API Key，所有数据存储于浏览器本地。
